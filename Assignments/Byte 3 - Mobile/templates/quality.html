Quality: About This Data Set

The quality of the data set is relatively high. Although it was collected every several minutes, it does not matter a lot when you look at locations of each hours. The data makes sense relative to itself and it is correct.

Is the data complete?

In a sense, the data is incomplete, because it only contains locations data during six days. Sampling bias may raise when researchers want to look into a more general location pattern. However, within the dataset, it is complete because there is no N/A or null values in any of the cells.

Is the data coherent?

The data makes sence relative to itself. The data is coherent if you look at the location of each hour. However, since the data was collected for every several mintues, there are some gaps between each data point. In this way the paths between locations may be somehow inaccurate.

Is the data correct?

The data is correct, because it clearly and accurately shows the locations I visited during the corresponding period.
